Assicurati di avere installato i seguenti strumenti sul tuo sistema:

## Installazione

Clona questo repository sul tuo sistema locale:
   ```bash
   git clone https://github.com/julioformiga/hack_attack.git
   cd hack_attack/thenoisycheckers
   docker-compose up
   ```
